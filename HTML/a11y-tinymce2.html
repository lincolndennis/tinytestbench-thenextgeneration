Introduction

TinyMCE’s new accessibility checker (based on technology built by Ephox) makes checking for common accessibility issues incredibly easy for content creators, using an interface that steps them through each error and in most cases also recommends a solution.

Learn more about accessibility in TinyMCE on our blog.

Accessibility highlight

* User dialog recommends fixes to common errors.

* Checks 12 WCAG & Section 508 accessibility issues.

* Familiar experience for most people, requiring little to no training.
